Project Description

This project was a simple responsive online wedding invitation created with HTML and CSS. The elements on the website were designed with the clients and it contained information about their wedding, such as time, location, dress code etc. The invitation was distributed via link and it was especially important that the mobile version of the website was created as it was expected that most guests would prefer it. I experimented with different versions and styles, and I especially played with the wedding typography aesthetic. (Personal information blurred and the website image is a stock image).
